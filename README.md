# Simple Interest Calculator

## Description

This project is a simple interest calculator developed using Bash scripting. It calculates the simple interest based on the principal amount, annual rate of interest, and time period entered by the user.

## Formula Used

Simple Interest = (Principal × Rate × Time) / 100

## Features

* Calculates simple interest quickly and accurately.
* Accepts user input for principal, rate, and time.
* Displays the calculated simple interest.
* Lightweight and easy to run on Linux or Unix systems.

## How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/Shreya-krishik/vftvk-Simple-Interest-Calculator.git
   ```

2. Navigate to the project directory:

   ```bash
   cd vftvk-Simple-Interest-Calculator
   ```

3. Make the script executable:

   ```bash
   chmod +x simple-interest.sh
   ```

4. Run the script:

   ```bash
   ./simple-interest.sh
   ```

## Example

**Input**

* Principal: 1000
* Rate: 5
* Time: 2

**Output**

```
Simple Interest = 100
```
